## How to use in your project?
SATIS

    php composer.phar create-project composer/satis --stability=dev --keep-vcs


create packages.json

    {
        "repositories": [
            {
                "type": "composer",
                "url": "https://php.letjson.com"
            }
        ]
    }

command:

    composer install
